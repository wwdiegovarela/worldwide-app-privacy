Preguntas Frecuentes

  • No puedo iniciar sesión: Verifica email/contraseña y conexión a Internet. Si el problema persiste, usa "¿Olvidaste tu contraseña?" para restablecerla.
  • No recibo notificaciones: En Android ve a Configuración → Aplicaciones → Worldwide → Notificaciones y habilítalas. En iOS ve a Ajustes → Notificaciones → Worldwide y habilítalas.
  • No veo mis instalaciones: Cierra sesión y vuelve a entrar para refrescar permisos. Si el problema persiste, contacta al administrador.
  • No puedo enviar mensajes: Verifica que tengas conexión a Internet y que la conversación tenga participantes. Si el problema persiste, intenta cerrar y reabrir la conversación.
  • No puedo ver archivos adjuntos: Verifica que tengas conexión a Internet. Los archivos se descargan desde Firebase Storage. Si el problema persiste, contacta al soporte.
  • No veo el estado online/offline de otros usuarios: Esta información se actualiza en tiempo real. Si no aparece, verifica tu conexión a Internet.
  • No puedo crear una conversación: Verifica que hayas seleccionado al menos una instalación y que hayas ingresado un título. El título es obligatorio.
  • No puedo cambiar la visibilidad de un mensaje: Esta función solo está disponible para usuarios con roles administrativos (Subgerente WFSA, Administrador WFSA).

Solución de Problemas

  1. Actualiza la app: Descarga la última versión desde Google Play Store (Android) o App Store (iOS).
  2. Reinicia el dispositivo: A veces un reinicio simple resuelve problemas de conexión o rendimiento.
  3. Verifica permisos:
    • Android: Configuración → Aplicaciones → Worldwide → Permisos (Notificaciones, Almacenamiento para adjuntos)
    • iOS: Ajustes → Worldwide → Notificaciones, Fotos (si quieres compartir imágenes)
  4. Limpia la caché: En Android, ve a Configuración → Aplicaciones → Worldwide → Almacenamiento → Limpiar caché.
  5. Cierra sesión y vuelve a entrar: Esto refresca tus permisos y sincroniza tu información con Firestore.
  6. Verifica tu conexión a Internet: La app requiere conexión a Internet para funcionar correctamente, especialmente para mensajería.
  7. Si persiste, reinstala la app: Desinstala y vuelve a instalar la aplicación. Nota: Esto eliminará la caché local, pero tus datos en la nube se mantendrán.

Problemas Específicos de Mensajería

  • Los mensajes no se envían: Verifica tu conexión a Internet. Si el problema persiste, intenta cerrar y reabrir la conversación.
  • No puedo subir imágenes: Verifica que hayas dado permisos de acceso a fotos/cámara. Las imágenes se comprimen automáticamente antes de subirse.
  • No puedo abrir documentos: Verifica tu conexión a Internet. Los documentos se descargan desde Firebase Storage. Si el problema persiste, contacta al soporte.
  • No veo mensajes nuevos: Desliza hacia abajo para refrescar la conversación. Si el problema persiste, cierra y reabre la app.
  • El contador de mensajes no leídos no se actualiza: Esto puede ocurrir si hay mensajes no visibles para tu rol. Los clientes solo ven mensajes marcados como "visibles para clientes".
